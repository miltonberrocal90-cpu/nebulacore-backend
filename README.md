# NebulaCore Backend

Este repositorio contiene el backend de NebulaCore, desarrollado con Node.js y Express.  
Incluye 17 módulos para gestión de empleados, roles, permisos y más.

## Tecnologías
- Node.js
- Express
- MongoDB (o la base que uses)
- JWT

## Endpoints principales
- `/api/empleados`
- `/api/roles`
- `/api/permisos`
